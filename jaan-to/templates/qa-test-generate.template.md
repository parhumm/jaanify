## CI Execution Guide

### Running Tests Locally

```bash
# Unit tests
npx vitest run --workspace=unit

# Integration tests (requires test DB)
npx vitest run --workspace=integration

# E2E tests
npx playwright test

# Smoke tests only
npx vitest run --workspace=unit -- --grep @smoke
npx playwright test --grep @smoke

# All BDD tests
npx vitest run --workspace=bdd-unit
npx vitest run --workspace=bdd-integration
npx playwright test --grep @e2e
```
