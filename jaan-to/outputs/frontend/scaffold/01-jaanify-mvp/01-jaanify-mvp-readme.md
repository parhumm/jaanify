## Prerequisites

- **Node.js** >= 22.0.0 LTS
- **pnpm** >= 9.0.0 (`corepack enable && corepack prepare pnpm@latest`)
- **Backend API** running at `http://localhost:3000` (see backend scaffold)

---

## Quick Start

```bash
# 1. Create Next.js project
pnpm create next-app jaanify-web --typescript --tailwind --eslint --app --use-pnpm --no-src-dir

# 2. Move into project
cd jaanify-web

# 3. Install dependencies
pnpm add react@^19 react-dom@^19 next@^15 @tanstack/react-query@^5 zustand@^5 nuqs@^2.5 next-themes@^0.4 clsx@^2.1 tailwind-merge@^2.6 zod@^3.24 axios@^1.7

# 4. Install dev dependencies
pnpm add -D typescript@^5.7 @types/react@^19 @types/react-dom@^19 @types/node@^22 tailwindcss@^4 @tailwindcss/postcss@^4 eslint@^9 eslint-config-next@^15 eslint-plugin-jsx-a11y@^6.10 prettier@^3.4 orval@^7 vitest@^2 @testing-library/react@^16 @testing-library/jest-dom@^6.6 jsdom@^25 @playwright/test@^1.50 @tanstack/react-query-devtools@^5

# 5. Copy scaffold files into project structure (see File Map below)

# 6. Configure environment
cp .env.local.example .env.local
# Edit .env.local with your API URL and Google OAuth client ID

# 7. Run development server
pnpm dev
```
