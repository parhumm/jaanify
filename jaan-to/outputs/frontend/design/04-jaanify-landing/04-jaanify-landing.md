## Design Rationale

### Aesthetic Choices

- **Sage/cream/terracotta palette** — Reuses existing design tokens from `globals.css` for visual consistency with dashboard and onboarding
- **DM Sans typeface** — Already loaded in `layout.tsx`, uses `clamp()` for fluid scaling
- **Asymmetric hero layout** — Text left + reasoning card right on desktop, stacked on mobile
- **Diagonal section divider** — CSS `clip-path` creates visual flow between hero and features
- **Soft geometric shapes** — Subtle background circles instead of stock photos or AI-generated images
- **SVG-only icons** — Inline SVGs matching the sage/terracotta palette, no icon library dependency

**Why these choices?**
The landing page is the first impression for productivity users. The warm palette and generous whitespace communicate calm competence — the opposite of overwhelming feature-heavy task managers. The reasoning card in the hero immediately demonstrates the USP (transparent AI reasoning).

**Differentiation from generic patterns:**
No hero gradient, no generic illustration, no blue/purple SaaS palette. The reasoning card preview acts as both visual interest and product demo. The diagonal divider creates asymmetry that breaks the standard centered-section-stack pattern.

### Technical Approach

- **Layout**: Flexbox-based sections with `max-w-5xl` container, CSS grid for feature cards and steps
- **Typography**: DM Sans with `clamp()` fluid scaling (2.25rem–3.75rem hero, 1.75rem–2.5rem sections)
- **Color**: CSS custom properties from `globals.css`, oklch color space with dark mode support
- **Motion**: IntersectionObserver-driven fade-in (0.7s ease-out), `prefers-reduced-motion` respected
- **Responsive**: Mobile-first (360px+), sm breakpoint (640px), desktop implicit via fluid typography

## Accessibility

**WCAG Compliance**: AA

| Aspect | Implementation |
|--------|----------------|
| Keyboard Navigation | Full tab order through nav, CTAs, links. Skip-to-content link |
| Screen Reader | ARIA labels on nav, sections via `aria-labelledby`, decorative elements hidden with `aria-hidden` |
| Color Contrast | Sage on cream ≥ 4.5:1 (checked via oklch), text on background ≥ 7:1 |
| Focus Indicators | `focus-visible:ring-2` on all interactive elements, never removed |
| Reduced Motion | `prefers-reduced-motion` media query disables all animations |
| Semantic HTML | `nav`, `section`, `article`, `footer`, `blockquote` — no div soup |
